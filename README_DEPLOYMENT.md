## 🚀 Quick Start (3 Steps)

### Step 1: Convert Graphics (10 minutes)
```
1. Go to https://svgtopng.com/
2. Upload assets/icon.svg
3. Download as icon.png (1024x1024)
4. Repeat for adaptive-icon.png and splash-icon.png
```

### Step 2: Download Assets (2 hours)
```
1. 3D Models: https://sketchfab.com/ (CC0 license)
2. Audio: https://freesound.org/ (CC0 license)
3. Follow ASSET_DOWNLOAD_GUIDE.md for direct links
```

### Step 3: Deploy (2 hours)
```bash
npm install -g eas-cli
eas login
eas build --platform all
eas submit --platform all
```
