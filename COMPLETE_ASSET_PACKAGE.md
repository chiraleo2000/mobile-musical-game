## 🎯 What You Need to Do (2-4 hours)

### Quick Path (2 hours minimum)

**Step 1: Convert Graphics (10 min)**
```
1. Go to https://svgtopng.com/
2. Upload assets/icon.svg → Download as icon.png (1024x1024)
3. Upload assets/icon.svg → Download as adaptive-icon.png (1024x1024)
4. Upload assets/splash.svg → Download as splash-icon.png (1284x2778)
```

**Step 2: Download 5 3D Models (1 hour)**
```
1. Go to https://sketchfab.com/
2. Search "xylophone" → Filter CC0 → Download as ranat-ek-medium.glb
3. Search "drum kit" → Filter CC0 → Download as drums-medium.glb
4. Search "guitar" → Filter CC0 → Download as guitar-medium.glb
5. Search "piano" → Filter CC0 → Download as piano-medium.glb
6. Search "xylophone" → Filter CC0 → Download as xylophone-medium.glb
```

**Step 3: Download 20 Audio Samples (30 min)**
```
1. Go to https://freesound.org/
2. Search each instrument → Filter CC0 → Download 4-5 samples each
3. Organize in assets/audio/ folders
```

**Step 4: Take 2 Screenshots (20 min)**
```
1. Run: npm start
2. Press 'i' for iOS or 'a' for Android
3. Take 2 screenshots
4. Save to screenshots/ folder
```
